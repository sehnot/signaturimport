SignaturImport.de

Importieren Sie Email-Signaturen als Kontakte in Ihr Adressbuch.

Die Herausforderung

Signaturen haben kein einheitliches Format

Jede E-Mail-Signatur sieht anders aus. Adressbuch-Apps können sie deshalb nicht direkt importieren – ein Zwischenschritt ist zwingend nötig.

Manuelle Eingabe kostet Zeit

Kontaktdaten aus E-Mail-Signaturen müssen mühsam von Hand ins Adressbuch übertragen werden – fehleranfällig und bei vielen Kontakten kaum zumutbar.

Online-Dienste übertragen Ihre Daten

Viele Konverter laden Signaturen auf fremde Server – ein inakzeptables Risiko, wenn Geschäftskontakte und persönliche Daten im Spiel sind.

Die Vorteile von SignaturImport.de

Datenschutz und Privatsphäre

Alle Daten bleiben auf Ihrem Gerät – es wird nichts ins Internet übertragen.

Offline verwendbar

Für maximalen Datenschutz können Sie SignaturImport herunterladen und vollständig ohne Internetverbindung nutzen.

Open Source

Der Quellcode ist öffentlich einsehbar und frei verwendbar. Das Repository ist auf GitHub verfügbar.

So funktioniert's

Schritt 01

Signatur einfügen

Kopieren Sie die E-Mail-Signatur und fügen Sie sie in das Textfeld ein.

Schritt 02

Daten analysieren

Die Kontaktdaten werden sofort automatisch erkannt und in die Felder eingetragen.

Schritt 03

Kontakt importieren

Laden Sie die extrahierten Daten als vCard herunter und importieren Sie den Kontakt in Ihr Adressbuch.
